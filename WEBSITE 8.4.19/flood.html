Floods Information

Flood Data is displayed in five different categories,
County - Showing the area of the flood in England.
Description - A small description of the flood, If undefined is shown then there is no description of the flood at current time.
Severity Level - How serious the flood is.
Is it tidal? - True stating that the water is still rising and false stating that it is not.
Severity level number - A number range of one to four stating how bad the flood is four being very serious and one being the least serious.
